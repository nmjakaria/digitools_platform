# 🛠️ DigiTools Platform

A modern, high-performance web platform designed to streamline digital workflows for creators and professionals. Built with a focus on clean UI/UX and seamless data handling.

[**Live Demo**](https://digitools-platform-by-nmjakaria.netlify.app/) | [**Sourse Code**](https://github.com/nmjakaria/digitools_platform.git)

-----

## 📝 Description

**DigiTools Platform** is a comprehensive solution for managing digital resources and professional workflows. This project features a highly responsive pricing architecture, a vibrant call-to-action interface, and a robust data-driven components system. It is designed to provide a "Software as a Service" (SaaS) experience, focusing on scalability and user engagement.

-----

## 🚀 Technologies Used

This project leverages a modern tech stack to ensure performance and maintainability:

  * **Frontend Library:** React.js
  * **Styling:** Tailwind CSS (Utility-first CSS) daisyUI (Faster, cleaner, easier
Tailwind CSS development)
  * **Icons:** [Lucide React](https://lucide.dev/), [React Icons](https://react-icons.github.io/react-icons/)
  * **Notifications:** [React Toastify](https://www.npmjs.com/package/react-toastify) (Customized alerts)
  * **Deployment:** Netlify
  * **Data Handling:** JSON-based dynamic rendering

-----

## ✨ Key Features

### 1\. Dynamic Pricing Engine

The platform includes a smart pricing component that renders plans dynamically from a JSON array. It features a "Most Popular" highlight logic that automatically adjusts the visual hierarchy, scaling the preferred plan and providing distinct styling for better conversion rates.

### 2\. Responsive "Glassmorphism" UI

The user interface is built with a mobile-first approach. It utilizes advanced Tailwind CSS techniques including backdrop filters, custom gradients, and flex-stretch grids to ensure all cards and sections maintain uniform height and alignment across all device sizes.

### 3\. Interactive Call-To-Action (CTA)

A high-conversion CTA section featuring a vibrant multi-color gradient background and micro-interactions. The buttons include hover-lift effects and active-state scaling, providing immediate tactile feedback to users.

## 🛠️ Installation & Setup

1.  **Clone the repository:**

    ```bash
    git clone https://github.com/nmjakaria/digitools_platform.git
    ```

2.  **Install dependencies:**

    ```bash
    npm install
    ```

3.  **Start the development server:**

    ```bash
    npm run dev
    ```
